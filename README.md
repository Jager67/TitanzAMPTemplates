# Titanz.tv AMP Templates
Custom AMP (Application Management Panel) templates maintained by Titanz.tv for game server hosting.

## About
This repository contains game server templates for use with CubeCoders AMP. These templates are customized and maintained by Titanz.tv to provide optimized configurations for our hosting infrastructure.

## Using These Templates
To use these templates in your AMP installation:

1. Navigate to Configuration > Application Settings > Configuration Repositories
2. Add this repository URL: `https://github.com/Jager67/TitanzAMPTemplates.git`
3. Save and refresh your instance list

## Making Generic Module Templates
See the official wiki article: https://github.com/CubeCoders/AMP/wiki/Configuring-the-'Generic'-AMP-module

Online configuration tools:
- Official: https://config.getamp.sh/
- Community (Beta): https://iceofwraith.github.io/GenericConfigGen/

## Template Guidelines
Templates in this repository follow these standards:

- No login/authentication required for downloads (except SteamCMD)
- Linux support via Proton when native builds unavailable
- Settings Manifest for customizable applications
- Console output exposure for AMP monitoring
- Direct executable launches only (no shell scripts)
